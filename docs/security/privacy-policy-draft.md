# Политика конфиденциальности PFM Bot

**Версия**: 1.0 Draft
**Дата**: 2026-03-20
**Статус**: Черновик — не опубликован официально

---

## 1. Кто мы

PFM Bot — сервис управления личными финансами, доступный через Telegram Mini App.
Сайт: mytodaylimit.ru
Контакт: [указать email или Telegram]

---

## 2. Какие данные мы собираем

### Данные из Telegram (при первом входе):
- Telegram ID (числовой, уникальный)
- Имя пользователя (first_name)
- Код языка (language_code)
- Telegram Chat ID (для отправки уведомлений через бота)

### Данные, которые вы вводите:
- Суммы доходов и даты выплат
- Суммы обязательных расходов
- Информация о кредитах/долгах (название, остаток, процентная ставка, минимальный платёж)
- Сумма подушки безопасности (текущая и целевая)
- Суммы трат (расходы в приложении)
- Часовой пояс
- Настройки уведомлений (время, включено/выключено)

### Технические данные:
- Дата и время входа (auth_date из Telegram)
- История периодов и ежедневных снимков (DailySnapshot)

### Что мы НЕ собираем:
- Email-адрес
- Телефон
- Фамилию или полное имя
- Фото профиля
- Геолокацию
- IP-адрес (не хранится в базе данных)
- Данные банковских карт (оплата через Telegram Stars — данные карты хранит Telegram, не мы)
- Историю переписки в Telegram

---

## 3. Как мы используем данные

- Расчёт лимита "Можно сегодня" и уведомления
- Отображение истории трат внутри периода
- Аналитика долговых обязательств (план лавины)
- Расчёт взноса в подушку безопасности
- Отправка уведомлений через Telegram Bot API

Данные НЕ используются для:
- Рекламы или ретаргетинга
- Передачи третьим лицам
- Машинного обучения на пользовательских данных
- Продажи или монетизации пользовательских данных

---

## 4. Аналитика

В настоящее время аналитические события не собираются. Никакие SDK аналитики (PostHog, Mixpanel и т.п.) не подключены. Cookies не используются.

При внедрении аналитики в будущем политика будет обновлена заранее.

---

## 5. Логи сервера

Сервер ведёт стандартные технические логи (метод запроса, путь, статус-код). Логи не содержат содержимое тел запросов (суммы расходов, финансовые данные). Логи хранятся в памяти контейнера и перезаписываются при перезапуске. Постоянного хранения логов не ведётся.

## 7. Передача данных третьим лицам

Данные передаются только:
- **Telegram** — для отправки уведомлений через Bot API (только Chat ID и текст сообщения). Регулируется политикой конфиденциальности Telegram: https://telegram.org/privacy
- **Telegram Stars** — при оплате подписки (обрабатывается Telegram, не нами). Мы получаем только подтверждение успешного платежа (Telegram Charge ID). Данные карты мы не получаем.

Никаким другим третьим лицам данные не передаются.

---

## 8. Срок хранения данных

Данные хранятся, пока пользователь активен — то есть пока аккаунт не удалён. Завершённые расчётные периоды и история расходов хранятся для отображения аналитики (экран "Прошлый период"). Автоматического удаления старых данных нет.

---

## 9. Удаление данных

**ВАЖНО**: Функция удаления данных по запросу пользователя в настоящее время **не реализована** (GAP-008, TD-007).

Для удаления данных: обратитесь напрямую к администратору через [Telegram/email].
После получения запроса все данные, связанные с вашим Telegram ID, будут удалены из базы данных в течение 3 рабочих дней. После удаления восстановление невозможно.

Планируется: команда /deletedata в боте.

---

## 10. Права пользователя

Вы вправе:
- Запросить перечень хранимых данных о вас (обратитесь к администратору)
- Запросить удаление данных (обратитесь к администратору, пока команда /deletedata не реализована)
- Запросить исправление некорректных данных
- Отключить уведомления в настройках приложения

Для реализации прав обратитесь к разработчику через [Telegram/email].

---

## 11. Оплата (Telegram Stars)

Подписка PFM PRO оплачивается через встроенный механизм Telegram Stars.

- Мы создаём ссылку на оплату через Telegram Bot API
- Платёж обрабатывается полностью на стороне Telegram
- Мы получаем только подтверждение успешного платежа (Telegram Charge ID и количество Stars)
- Мы НЕ получаем данные карты, счёта или любой платёжной информации

Возврат Stars: регулируется правилами Telegram. Обращайтесь в поддержку Telegram.

---

## 12. Безопасность

- Соединение с сервером защищено SSL/TLS (HTTPS, Let's Encrypt)
- API доступен только через HTTPS; прямой доступ к базе данных из интернета закрыт
- Авторизация построена на проверке подписи Telegram (HMAC-SHA256)
- Токен бота хранится в защищённой переменной окружения, не в коде
- Время действия токена авторизации ограничено 1 часом (защита от повторного использования)

## 13. Изменения политики

О существенных изменениях будет сообщено через бот-уведомление. Продолжение использования сервиса после уведомления означает согласие с обновлённой политикой.

Настоящая политика регулируется законодательством Российской Федерации, в том числе Федеральным законом № 152-ФЗ «О персональных данных».
