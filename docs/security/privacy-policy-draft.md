# Политика конфиденциальности — PFM Bot / MyTodayLimit

---

## 1. Кто мы

Приложение «MyTodayLimit» (Telegram Mini App и бот) — персональный финансовый трекер.
Сервис доступен по адресу: https://mytodaylimit.ru и через Telegram.

Контакт разработчика: Через бота: @[BOT_USERNAME] *(точный username уточняется)*

---

## 2. Какие данные мы собираем

Мы собираем только те данные, которые необходимы для работы сервиса.

### Данные, которые мы получаем от Telegram:

| Поле | Зачем |
|------|-------|
| Telegram User ID (числовой) | Идентификация пользователя в нашей базе данных |
| Имя (first_name) из профиля Telegram | Отображение в интерфейсе приветствия |
| Telegram Chat ID | Отправка уведомлений через бота |

### Данные, которые вводит пользователь:

| Данные | Зачем |
|--------|-------|
| Сумма дохода и дни выплаты | Расчёт лимита на день (S2S) |
| Обязательные платежи (аренда, кредит и т.д.) | Вычет из бюджета при расчёте S2S |
| Долги (остаток, ставка, минимальный платёж) | Расчёт плана погашения (метод лавины) |
| Текущий и целевой размер подушки безопасности | Расчёт ежемесячного взноса в резервный фонд |
| Расходы (сумма и опциональная заметка) | Учёт трат для расчёта остатка на день |
| Часовой пояс | Отправка уведомлений в нужное время |
| Настройки уведомлений (время, включено/выключено) | Рассылка утренних и вечерних напоминаний |

---

## 3. Что мы НЕ собираем

- Номер телефона
- Email-адрес
- Фамилию или полное имя
- Фото профиля
- Геолокацию
- Данные банковских карт (платежи через Telegram Stars обрабатываются Telegram, мы не получаем платёжные данные)
- Историю переписки в Telegram

---

## 4. Аналитика

В настоящее время аналитические события не собираются. Никакие SDK аналитики (PostHog, Mixpanel и т.п.) не подключены.

В будущем могут быть добавлены агрегированные метрики использования (без персональных данных) для улучшения сервиса. При внедрении аналитики политика будет обновлена.

---

## 5. Логи сервера

Сервер ведёт стандартные логи HTTP-запросов (метод, путь, статус-код, timestamp). Логи **не содержат** содержимое тел запросов (суммы расходов, заметки, финансовые данные).

Логи хранятся в памяти контейнера и автоматически перезаписываются при перезапуске контейнера. Постоянного хранения логов не ведётся.

---

## 6. Обработка IP-адреса

IP-адрес пользователя обрабатывается reverse-proxy (nginx) для базовой защиты от флуда и DDoS. IP-адреса не хранятся в базе данных и не связываются с профилем пользователя.

---

## 7. Где хранятся данные

Все данные хранятся на выделенном сервере в России:
- Провайдер: Timeweb Cloud
- Страна: Россия
- Сервер: VPS (Честный Ворон)

База данных: PostgreSQL. Доступ к серверу ограничен — только SSH с авторизованного устройства разработчика.

---

## 8. Как используются данные

Ваши данные используются исключительно для:

1. Расчёта персонального финансового плана (лимит на день, план погашения долгов,
   взнос в подушку безопасности).
2. Отображения истории расходов в приложении.
3. Отправки уведомлений через Telegram-бота (если уведомления включены).
4. Смены расчётного периода в автоматическом режиме.

Данные не используются для таргетированной рекламы, профилирования, продажи или передачи третьим лицам.

---

## 9. Передача третьим лицам

Мы НЕ передаём ваши данные третьим лицам.

Единственное исключение: Telegram (необходим для доставки уведомлений и авторизации). Telegram получает только то, что требуется для отправки сообщения: Chat ID и текст уведомления. Это регулируется политикой конфиденциальности Telegram: https://telegram.org/privacy

---

## 10. Срок хранения данных

Данные хранятся, пока пользователь активен — то есть пока аккаунт не удалён.

Завершённые расчётные периоды и история расходов хранятся для отображения аналитики (экран «Прошлый период»). Автоматического удаления старых данных нет.

---

## 11. Удаление данных

Вы можете запросить удаление всех ваших данных.

**Как удалить:**

Команда `/delete` в боте находится в разработке. На данный момент данные удаляются по письменному запросу: напишите боту @[BOT_USERNAME] с текстом «Удали мои данные».

После получения запроса все данные, связанные с вашим Telegram ID, будут удалены из базы данных в течение 3 рабочих дней. После удаления восстановление невозможно.

---

## 12. Оплата (Telegram Stars)

Подписка PFM PRO оплачивается через встроенный механизм Telegram Stars.

- Мы создаём ссылку на оплату через Telegram Bot API.
- Платёж обрабатывается полностью на стороне Telegram.
- Мы получаем только подтверждение успешного платежа (`successful_payment`): Telegram Charge ID и количество Stars.
- Мы НЕ получаем данные карты, счёта или любой платёжной информации.

Возврат Stars: регулируется правилами Telegram. Обращайтесь в поддержку Telegram.

---

## 13. Безопасность

- Соединение с сервером защищено SSL/TLS (HTTPS).
- API доступен только через HTTPS. Прямой доступ к базе данных из интернета закрыт.
- Авторизация в приложении построена на проверке подписи Telegram (HMAC-SHA256).
- Токен бота хранится в защищённой переменной окружения, не в коде.
- Время действия токена авторизации ограничено 1 часом.

---

## 14. Права пользователя

В соответствии с применимым законодательством вы имеете право:

- Запросить перечень хранимых данных о вас.
- Запросить исправление некорректных данных.
- Запросить удаление данных (см. раздел 11).
- Возразить против обработки данных.

Для реализации прав обратитесь к разработчику через бота: @[BOT_USERNAME]

---

## 15. Изменения политики

При существенных изменениях в политике конфиденциальности уведомление будет опубликовано в боте. Продолжение использования сервиса после уведомления означает согласие с обновлённой политикой.

---

## 16. Применимое право

Настоящая политика регулируется законодательством Российской Федерации, в том числе Федеральным законом № 152-ФЗ «О персональных данных».
