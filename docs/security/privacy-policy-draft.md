## 2. Какие данные мы собираем

Мы собираем только те данные, которые необходимы для работы сервиса.

### Данные, которые мы получаем от Telegram:

| Поле | Зачем |
|------|-------|
| Telegram User ID (числовой) | Идентификация пользователя в нашей базе данных |
| Имя (first_name) из профиля Telegram | Отображение в интерфейсе приветствия |
| Telegram Chat ID | Отправка уведомлений через бота |

### Данные, которые вводит пользователь:

| Данные | Зачем |
|--------|-------|
| Сумма дохода и дни выплаты | Расчёт лимита на день (S2S) |
| Обязательные платежи (аренда, кредит и т.д.) | Вычет из бюджета при расчёте S2S |
| Долги (остаток, ставка, минимальный платёж) | Расчёт плана погашения (метод лавины) |
| Текущий и целевой размер подушки безопасности | Расчёт ежемесячного взноса в резервный фонд |
| Расходы (сумма и опциональная заметка) | Учёт трат для расчёта остатка на день |
| Часовой пояс | Отправка уведомлений в нужное время |
| Настройки уведомлений (время, включено/выключено) | Рассылка утренних и вечерних напоминаний |

---

## 3. Что мы НЕ собираем

- Номер телефона
- Email-адрес
- Фамилию или полное имя
- Фото профиля
- Геолокацию
- Данные банковских карт (платежи через Telegram Stars обрабатываются Telegram, мы не получаем платёжные данные)
- Историю переписки в Telegram
- IP-адрес пользователя

## 5. Как используются данные

Ваши данные используются исключительно для:

1. Расчёта персонального финансового плана (лимит на день, план погашения долгов,
   взнос в подушку безопасности).
2. Отображения истории расходов и аналитики в приложении.
3. Отправки уведомлений через Telegram-бота (если уведомления включены).
4. Смены расчётного периода в автоматическом режиме.

Данные не используются для таргетированной рекламы, профилирования, продажи или
передачи третьим лицам.

---

## 6. Передача третьим лицам

Мы НЕ передаём ваши данные третьим лицам.

Единственное исключение: Telegram (необходим для доставки уведомлений и авторизации).
Telegram получает только то, что требуется для отправки сообщения: Chat ID и текст
уведомления. Это регулируется политикой конфиденциальности Telegram:
https://telegram.org/privacy

---

## 7. Срок хранения данных

Данные хранятся, пока пользователь активен — то есть пока аккаунт не удалён.

Завершённые расчётные периоды и история расходов хранятся для отображения аналитики
(экран «Прошлый период»). Автоматического удаления старых данных нет.

## 9. Оплата (Telegram Stars)

Подписка PFM PRO оплачивается через встроенный механизм Telegram Stars.

- Мы создаём ссылку на оплату через Telegram Bot API.
- Платёж обрабатывается полностью на стороне Telegram.
- Мы получаем только подтверждение успешного платежа (`successful_payment`):
  Telegram Charge ID и количество Stars.
- Мы НЕ получаем данные карты, счёта или любой платёжной информации.

Возврат Stars: регулируется правилами Telegram. Обращайтесь в поддержку Telegram.

## 10. Безопасность

- Соединение с сервером защищено SSL/TLS (HTTPS).
- API доступен только через HTTPS. Прямой доступ к базе данных из интернета закрыт.
- Авторизация в приложении построена на проверке подписи Telegram (HMAC-SHA256).
- Токен бота хранится в защищённой переменной окружения, не в коде.
